3. 케릭터 선택화면
   - 격투 게임 케릭터 선택과 같이 선택 가능한 케릭터 들이 작은 사진으로 바 형식으로 나열되어있음
   - 1p 라고 적힌 네모박스가 위의 케릭터들위에 한개 떠서 키보드 움직일때 박스가 위치 변경해서 
     z를 클릭 또는 마우스로 케릭터를 클릭하면 1P가 해당 케릭터로 확정
   - 1P 케릭터가 확정된 경우 선택바 왼쪽에 고른 케릭터의 풀사진을 뛰워주고 케릭터바에서 선택한 케릭터는 회색바로 고를수 없게 됨
   - CPU 라고 적힌 네모박스로 전환되고 동일하게 케릭터를 고르게 함

   - 케릭터 선택이 완료된 이후 배경을 선택하는 구성에서 동일하게 배경선택
   - 배경까지 고른경우 4. 게임화면 으로 이동

4. 게임 화면
   - 위에서 1/3 에 검정 바가 지나가고 아래는 오셀로 영역, 위는 격투게임 영역
   - 격투게임 영역을 위에서 아래로 5등분으로 나누고,  2/5 의 영역은 체력바 영역, 4/5는 케릭터 영역, 나머지는 격투게임 배경영역
   - 체력바 영역은 좌에서 우로 5등분 해서 2/5 부분은 1P 체력바 영역, 4/5 부분은 CPU 체력바 영역
   - 체력은 각각 10000P 로 구성
   - 케릭터 영역에서는 1P, CPU 로 고른 케릭터들이 서로 서있음
   - 오셀로 영역에서는 좌에서 우로 5등분 해서 1/5에는 1P 케릭터 스킬 버튼 (작은버튼 2개+큰버튼 1개) 5/5 에는 CPU 케릭터 스킬 버튼 (작은 버튼 2개 + 큰버튼 1개) 
   - 버튼에는 각각 케릭터 사진(케릭터 선택시에 사용한)이 떠있고, 정해진 파일에 있는 스킬명이 버튼 아래 떠잇음
   - 오셀로 영역의 스킬바를 제외한 영역에는 바둑판이 떠있음
   - 바둑판에는 오셀로 최초 규칙에 따라 돌이 놓여져 있음
   - 검은돌, 흰돌 대신 각각 고른 케릭터 사진이 첨부된 돌이 놓여져 있음

4-1. 게임 플레이
   - 1p 부터 번갈아서 바둑알을 바둑판에 둠
   - 오셀로 규칙대로 놓고 난 뒤 상대방 돌을 뒤집게 됨
   - 뒤집혀진 알 개수를 체크해서 상대방 체력을 감소
   - 뒤집혀진 알 개수가 5배수 마다 발생하는 이펙트, 그림, 소리가 다름
   - 돌을 놓지 않는 3초에 한번씩 나한태 데미지 1p가 들어옴
   - 데미지가 들어올때마다 케릭터 맞는 이펙트와 모션, 체력바 반짝임, 상대방 케릭터는 때리는 모션
   - 오셀로 규칙대로 돌을 놓을수 있음 
   - 상대방이 돌을 놓을수 없을때, 연속해서 돌을 놓는 차례가 오고 이떄 뒤집는 데미지는 1.5배
   - 위 연속이 계속되는만큼 1.5배 제곱, 이펙트, 소리
   - 한쪽의 체력이 0이하로 떨어지면 "K.O" 라는 이펙트, 박스가 떨어지고 3초마다 통통 튐
   - 위 상황에서 어떤 버튼이든 클릭시 3. 케릭터 선택 으로 이동 
   - 체력이 다 떨어지지 않았는데, 돌을 양쪽다 놓을수 없을때 오셀로 돌이 어느쪽이 많은지 확인하고 
     더 적은 측에 돌 차이 * 10P 의 체력 데미지, FINISH 라는 이펙트와 케릭터 모션
